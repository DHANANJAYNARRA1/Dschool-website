# Testimonial Frames Guide

## What are Testimonial Frames?

Branded testimonial images that you can use for:
- Social media posts
- Marketing materials
- Website testimonials section
- Print materials

## How to Generate Frames

### Step 1: Access the Generator
Visit: `http://localhost:5173/testimonial-frames`

### Step 2: Customize
Edit `/src/app/components/TestimonialFrames.tsx` and add your testimonials:

```typescript
const sampleTestimonials = [
  {
    quote: "Your testimonial text here",
    author: "Person Name",
    role: "Their Role/Position",
  },
  // Add more testimonials...
];
```

### Step 3: Download
1. Right-click on the frame
2. Select "Save Image As" or "Screenshot"
3. Use in your marketing materials

## Frame Features

✓ Branded with D School logo and colors
✓ Professional design with quote styling
✓ Author name and role
✓ Navy blue and gold color scheme
✓ Ready for social media (Instagram, LinkedIn, Facebook)

## Dimensions

- Default: 2:1 ratio (optimal for most platforms)
- Recommended for Instagram: 1080x1080px
- Recommended for LinkedIn: 1200x627px
- Recommended for Facebook: 1200x630px

## Tips

- Keep quotes concise (2-3 sentences max)
- Use high-quality photos if adding custom images
- Maintain consistent branding across all frames
- Test on different platforms before posting
